> 梁思成先生是我国著名的建筑学家与建筑教育家，他以毕生精力投入到古代建筑的研究工作之中，取得了世人瞩目的成就，他是中国建筑历史与理论学科的奠基人之一。
>
> 梁思成应用世界各国调查研究古代建筑先进之方法，自上个世纪三十年代开始对中国古代建筑进行了长时间的、科学的调查，他和他的同事们在不到十年的时间里对全国近200个县城的两千余个古建筑项目进行了考察，写出了一篇又一篇详尽的研究报告，积累了大量翔实的文字与图片资料。梁思成正是在这样的基础上，在抗日战争及其艰苦的条件下完成了《中国建筑史》的写作。这是中国第一部较为系统的论述我国古代建筑发展历史的专著。
> ——楼庆西

本文是Aaron阅读《中国建筑史》过程中的随记。

# 第一章 绪论

## 第一节 中国建筑之特征

**中国建筑之主要特征**<sup>[[1](#1.1)]</sup><sup>[[2](#1.2)]</sup><sup>[[3](#1.3)]</sup><sup>[[4](#1.4)]</sup>

**结构取法及发展**方面：

- 以木材为主要构材
- 历用架构制之结构原则<sup>[[5](#1.5)]</sup><sup>[[6](#1.6)]</sup><sup>[[7](#1.7)]</sup><sup>[[8](#1.8)]</sup><sup>[[9](#1.9)]</sup>
- 以斗栱为结构之关键，并为度量单位<sup>[[10](#1.10)]</sup><sup>[[11](#1.11)]</sup>
- 外部轮廓之特异：
    1. 翼展之屋顶部分<sup>[[12](#1.12)]</sup>
    2. 崇厚阶基之衬托
    3. 前面玲珑木质之屋身
    4. 院落之组织
    5. 彩色之施用
    6. 绝对匀称与绝对自由之两种平面布局<sup>[[13](#1.13)]</sup>
    7. 用石方法之失败<sup>[[14](#1.14)]</sup>

**环境思想**方面：

- 不求原物长存之观念
    1. 满足于木材之沿用
    2. 修葺原物之风，远不及重建之盛
- 建筑活动受道德观念之制裁
- 着重部署之规制
- 建筑之术，师徒传授，不重书籍

1. <span id="1.1">建筑之始，产生于实际需要，受制于自然物理</span>
2. <span id="1.2">其结构之系统及形式之派别，乃其材料环境所形成</span>
3. <span id="1.3">建筑活动与民族文化之动向实相牵连，互为因果</span>
4. <span id="1.4">建筑显著特征之所以形成，有两因素：有属于实物结构技术上之取法及发展者；有缘于环境思想之趋向者</span>
5. <span id="1.5">以立柱四根，上施梁枋，牵制成为一“间”</span>
6. <span id="1.6">梁可数层重叠称“梁架”</span>
7. <span id="1.7">逐级增高称“举折”</span>
8. <span id="1.8">建筑物上层之一切负荷均由架构负担；承重者为其立柱及其梁枋</span>
9. <span id="1.9">建筑物中所有墙壁，均为“隔断墙”，非负重之部分</span>
10. <span id="1.10">斗栱之功用在以伸出之栱承受上部结构之荷载，转纳于下部之立柱上，故为大建筑物所必用</span>
11. <span id="1.11">斗栱之组织与比例大小，历代不同，每可藉其结构演变之序，以鉴定建筑物之年代</span>
12. <span id="1.12">上尊而宇卑，则吐水疾而霤（音liù）远 <!-- 大致是说屋顶高而屋檐低，坡度大，利于排水。--></span>
13. <span id="1.13">绝对匀称与绝对自由之两种传统之平面部署，在不觉中，含蕴中国生活精神之各面，至为深刻 <!-- 行为上恪守规范，精神上自由无羁 --></span>
14. <span id="1.14">隋安济桥之建造者李春</span>
15. <span id="1.15">天宝乱后，及会昌、后周两次灭法，建筑精华毁灭殆尽</span>
16. <span id="1.16">自上古以至秦期间文献与实物双方资料皆极缺乏</span>
17. <span id="1.17">两汉时期为中国建筑发育时期，建筑事业极为活跃，史籍中关于建筑之记载颇为丰富，建筑之结构形状则有遗物可考其大略</span>
18. <span id="1.18">魏晋南北朝时期，宫殿与佛寺之建筑活动仍极为澎湃</span>
19. <span id="1.19">隋唐时期，宫殿、寺观之建筑均为活跃。唐之建筑风格，既以倔强粗壮胜，其手法又以柔和精美见长，诚蔚然大观</span>
20. <span id="1.20">五代、赵宋之后，中国之艺术开始华丽极致，至宋中叶以后乃趋纤靡文弱之势。宋辽金作风手法，特征显著，规例谨慎，循旧制之途径，增减嬗变不已</span>
21. <span id="1.21">元明清时期建筑传统仍一仍<!--依？-->古制。至明清之交，有西藏样式和西洋样式输入。清末，西洋建筑之样式渗入都市，旧建筑之势力日弱</span>
22. <span id="1.22">民国初年，建筑活动颇为停滞。近二十年来，建筑师始渐回顾及中国固有之建筑式样，对于其设计样式均有其地域或时代样式之自觉。今后之中国建筑，将减少其绝对之地方性</span>
23. <span id="1.23">《营造法式》，宋李诫著。诫，徽宗朝将作少监也。全书三十四卷。“大木作制度”为全书最重要部分</span>
24. <span id="1.24">材之大小共分八等，视建筑物之大小等第而定其用材之等第</span>
25. <span id="1.25">“各以其材之广，分为**十五**分，以**十**分为其厚”</span>
26. <span id="1.26">两材之间，以斗垫托其空隙，其空隙距离为六分，称为“**栔**”</span>
27. <span id="1.27">凡高一材一栔（即高二十一分）之材，谓之足材。</span>
28. <span id="1.28">宋式建筑各部间之比例，皆以其所用材之材“栔”分为度量标准</span>
29. <span id="1.29">斗栱由若干斗与栱垒叠而成，总称“铺作”。在柱头上者称“柱头铺作”，在柱与柱之间者称“补间铺作”，在角柱之上者称“转角铺作”</span>
30. <span id="1.30">铺作构材有斗、栱、昂三类，最重要者为集中全铺作重量之栌斗，及由栌斗向前后出跳之华栱</span>
31. <span id="1.31">栌斗中心及每跳跳头或施横栱，谓之计心；或不施横栱，谓之偷心</span>
32. <span id="1.32">横栱一层者为单栱，双层者为重栱</span>
33. <span id="1.33">殿阁如用平闇（即天花板），则平闇以下梁袱，谓之“明袱”。平闇之上另有草袱以承屋盖之重</span>
34. <span id="1.34">举折即定屋顶坡度及屋盖曲面线之方法也</span>
35. <span id="1.35">清式建筑的度量单位为斗口。斗口者宋式之材厚也。<!--宋式材厚十分-->斗栱比例亦以斗口之倍数或分数为准</span>
36. <span id="1.36">铺作之间的距离规定为十一斗口。因此柱之分配，柱间之距离，面阔进深之尺度，皆以两朵间距离十一斗口之倍数为准则焉</span>
37. <span id="1.37">建筑物立面上，檐柱自中央由当心间向两端依次升高，使檐口呈一缓和优美的曲线，这种做法在宋《营造法式》中称为“生起”</span>
38. <span id="1.38">清式角柱与平柱同高，且柱为直柱，无卷杀</span>
39. <span id="1.39">宋式梁枋断面高宽均为三与二之比。至清式则改为五与四或六与五之比</span>
40. <span id="1.40">清式屋盖较宋式屋盖陡峻</span>

# 第二章 上古时期

## 第一节 上古

- 中国建筑之原始，究起自何时，待将永远笼罩于史前之玄秘中
- 殷商以前，史难置信
- 至殷代末年（公元前十二世纪顷），纣王广作宫室，益广囿苑
- 周文王都丰、武王都镐。屋顶之如翼，木柱之采用，庭院之平正，已成定法
- 文王作灵囿，内有灵台、灵沼，为中国史传最古之公园
- 成王之时，周公“复营洛邑”。此为我国史籍中关于都市设计最古之实录<sup>[[1](#2.1)]</sup>

## 第二节 春秋战国

- 强大诸侯国营建渐侈
- 自天子至庶人，其宗庙建筑，均有一定制度
- 卿大夫之宅，规则尤严
- 当时盛游猎之风，故喜园囿
- 至战国之室，仅余七雄。对建筑之观念，不若前此之简朴<sup>[[2](#2.2)]</sup><sup>[[3](#2.3)]</sup>

## 第三节 秦

- 秦始皇统一天下，以孝公时代商鞅所营咸阳宫廷冀阙为核心而扩大增益之
- 营作朝宫渭南上林苑中
- 始皇陵墓建筑豪奢，亦前所未有
- 始皇筑长城

## 笔记

1. <span id="2.1">都市之制：天子都城：“方九里，旁三门。国中九经九纬，经涂九轨。左祖右社，面朝后市”</span>
2. <span id="2.2">诸侯均已“高台榭，美宫室”。且自周中世以降，尤尚殿基高巨之风</span>
3. <span id="2.3">万里长城，于战国之世亦已由各国分段兴筑</span>

# 第四章 魏、晋、南北朝

## 第一节 文献上魏、晋建筑之大略

- 汉末曹操居邺，治府第，作三台
- 魏文帝受汉禅（公元220年），营洛阳宫，初居北宫，以建始殿朝群臣
- 孙权都建邺，节俭不尚土木之功，至孙皓起昭明宫，始破坏诸营。刘备在蜀，营建较少
- 晋初仍魏，宫殿少有损益
- 东晋元帝立宗庙社稷于建康。元、明二帝亦不改制。成帝营“建平园为宫”。孝武帝改作新宫
- 晋室南迁，五代偏据交相替迭，各有营建，其中最为僭移、史传最详者，莫如后赵石氏（公元319-352年）
- 东晋之末，赫连勃勃营起统万城于今陕西横山县西之地。其营缮规模亦颇可观
- 佛教既入中国，至后汉末，佛寺佛塔之建筑，已行于全国<sup>[[1](#4.1)]</sup><sup>[[2](#4.2)]</sup>

## 第二节 南北朝之建筑活动

- 南朝宋、齐、梁、陈均都建康
- 齐代宫苑之侈，以东昏候为最
- 拓跋魏营建之功极盛
- 平城魏陵墓多建石室
- 自道武帝营平城，至孝文帝迁洛京，平城宫郭苑囿营建之见于史籍者尚极多
- 佛教至晋而普传中国，其在北魏则“京邑帝里，佛法丰盛，神图妙塔，桀跱相望，法轮东转，兹为上矣”
- 魏在代京武州川营窟寺
- 齐代对于长城颇加修筑
- 西魏都长按，少有营缮

## 笔记

1. <span id="4.1">至晋世而佛教普传，高僧辈出，寺塔林立</span>
2. <span id="4.2">晋恭帝“造丈六金像，亲迎于瓦棺寺”。孝武帝则“立精舍于殿内”，千数年灿烂辉煌之佛教建筑活动，至是已开始矣</span>
3. <span id="4.3">南朝宫殿、佛寺今无存者，陵墓石刻则南京丹阳附近遗物尚多</span>
4. <span id="4.4">沙门昙曜于北魏文成帝兴安二年（公元453年）,“凿山石壁，开窟五所，镌建佛像各一，高者七十尺，次六十尺。雕饰奇伟，冠于一世”</span>
5. <span id="4.5">现存大窟十九。壁龛无数</span>
6. <span id="4.6">昙曜所开五窟，在崖壁西部，其平面作椭圆形，佛像形制，最为古拙。洞中仅刻佛菩萨像，壁上无佛绩图或其他装饰</span>
7. <span id="4.7">魏既迁都洛阳，于景明元年（公元500年）营伊阙石窟，历时二十四载始成</span>
8. <span id="4.8">今称古阳洞及宾阳三洞者，即此时期所凿造</span>
9. <span id="4.9">北魏孝明帝正光元年（公元520年）建，为国内现存最古之砖塔</span>
10. <span id="4.10">塔本瘗佛骨之所，梵语曰“窣堵坡”（Stupa），译义为坟，冢，灵庙。其在印度大多为半圆球形冢，而上立刹者。及其传至中国，于汉末三国时代，“上累金盘，下为重楼”，殆即以印度之窣堵坡置于中国原有之重楼之上，遂产生南北朝所最通常之木塔</span>

# 生僻字

- 上尊而宇卑，则吐水疾而**霤**远。音liù。屋檐的流水。如: 霤水(屋檐水)
- 鲜焕者尚**夥**。音huǒ。（形声。从多，果声。本义：盛多） 同本义 [many;much]。如：获益甚夥
- **髹**漆。音xiū。用漆涂在器物上
- **嬗**变。音shàn。变迁、更替
- **栔**。音qì。古代木结构的一种术语，与斗栱结构有关。指的是上下栱之间填充的断面尺寸，和“材”一样，是建筑尺度的计量标准
- 平**闇**。音àn。天花板
- 梁**袱**。音fú。如：明袱，露在外面，由下面可以看见的梁袱
- **斫**（音zhuó。大锄；引申为用刀、斧等砍）其椽而**砻**（音lóng。磨，如:砻刻。磨光雕琢）之
- 桀**跱**。音zhì。耸立
- 壁**龛**。音kān。壁龛最早是在宗教的建筑上，基本都是在建筑物上凿进出一个空间
- **瘗**佛骨。音yì。掩埋，埋葬。如：瘗玉埋香
- 平**棊**。音qí。出自《营造法式·小木作制度三》，解释为大的方木格网上置板，并便施彩画的一种天花
- **盝**顶。音lù。盝顶是中国古代传统建筑的一种屋顶样式，顶部有四个正脊围成为平顶，下接庑殿顶

# 术语

- 一斗三升：由一只大斗、一个横栱和三个三才升构成的斗栱。属不出踩斗栱，只起传导荷载作用。是斗栱中最简单、最原始的一种
- 卷杀：卷杀是中国古人在做建筑时，将构件或部位的端部做成缓和的曲线或折线形式，使得构件或部位的外观显得丰满柔和，此为卷杀。”卷“有圆弧之意，“杀”有砍削之意
- 收分：中国古代的圆柱子上下两端直径是不相等的，除去瓜柱一类短柱外，任何柱子都不是上下等径的圆柱体，而是根部略粗，顶部略细，这种作法，称为“收溜”又称“收分”
- 反宇：屋檐上仰起的瓦头
- 翘角：中国传统建筑檐部形式之一，多指屋檐特别是屋角的檐部向上翘起，若飞举之势
- 发券：利用块料之间的侧压力建成跨空的承重结构的砌筑方法称发券
- 叠涩：叠涩是一种古代砖石结构建筑的砌法，用砖，石，有时也用木材通过一层层堆叠向外挑出，或收进，向外挑出时要承担上层的重量
